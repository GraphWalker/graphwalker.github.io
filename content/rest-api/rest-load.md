## Load
The Rest call `load` uploads model(s) in JSON format and resets GraphWalker with the new test.
 
**POST Request**

http://service-host:8887/graphwalker/load + json graph file

**Response**

If the request was successful **"result"** will be **"ok"**.

```json
{
  "result":"ok"
}
```
